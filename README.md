# tftp-rs

High performance Trivial File Transfer Protocol (TFTP) library for Rust

## RFCs implemented:

* [RFC 1350](https://tools.ietf.org/html/rfc1350)

## Contributing

### Getting the code

```
git clone https://github.com/arjantop/tftp-rs
cd tftp-rs
cargo build
```
